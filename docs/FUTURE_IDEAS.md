# Future Ideas

This file is for ideas that are interesting but not part of the current milestone.

The purpose of this file is to preserve excitement without derailing progress.

## AI Features

* Paste messy workout notes and have AI parse them into structured data
* Generate weekly training reviews
* Recommend the next workout based on recent history
* Flag possible overuse, pain, or recovery issues
* Summarize trends across months of workouts
* Generate “coach notes” after each workout

## Habit / Identity Features

* Track post-meal walks
* Track protein-first meals
* Track “identity wins”
* Track moments where I negotiated with myself
* Generate a weekly identity review
* Show evidence of consistency over time

## Exercise Features

* Exercise history pages
* Estimated 1 rep max calculations
* Personal records
* Volume tracking
* Graphs for major lifts
* Notes by exercise
* Tags for technique cues

## Deployment Ideas

* Deploy locally first
* Later deploy to Synology NAS
* Possibly deploy to Render, Fly.io, or another hosted provider
* Use Docker when ready

## Local AI Stretch Goal

Local AI is a stretch goal, not a current requirement.

Possible future tools:

* Ollama
* Open WebUI
* Local note parsing
* Local first-draft summaries

This should not be explored until the basic workout app is useful.
